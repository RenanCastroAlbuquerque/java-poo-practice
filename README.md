# Java POO Practice

Este repositório reúne meus estudos, exercícios e projetos práticos focados em Java e Programação Orientada a Objetos (POO).

O objetivo é fortalecer meu conhecimento nos fundamentos da linguagem Java e aplicar os princípios da orientação a objetos em códigos reais.

## O que você encontrará aqui

- Fundamentos de Java  
- Conceitos de Programação Orientada a Objetos  
- Exercícios práticos  
- Pequenos projetos  
- Desafios de lógica e resolução de problemas  

## Tópicos abordados

- Classes e Objetos  
- Encapsulamento  
- Herança  
- Polimorfismo  
- Abstração  
- Construtores  
- Métodos e Atributos  
- Collections  
- Tratamento de Exceções  

## Propósito

Este repositório documenta minha jornada de aprendizado em Java e serve como um portfólio da minha evolução como desenvolvedor de software.
